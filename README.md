# FollowUpFlow
 
A lightweight dashboard that helps students and junior job-seekers track
job applications, automatic follow-up dates, and recruiter notes.
 
## Problem
Applicants apply through many channels (LinkedIn, email, forms, referrals)
and lose track of status, follow-ups, and notes across memory, spreadsheets,
and chat apps. FollowUpFlow keeps everything in one focused view.
 
## Tech stack
- Next.js (App Router) + TypeScript
- Tailwind CSS
- React state + custom hooks
- Browser localStorage for persistence
 
## Local setup
```bash
git clone <your-repo-url>
cd followupflow
npm install
npm run dev
```
Open http://localhost:3000
